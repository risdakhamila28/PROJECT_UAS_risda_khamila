Profile

Perkenalkan, nama saya Risda Khamila. Saya merupakan anak kedua dari tiga bersaudara dan saat ini berusia 19 tahun. Saya berasal dari Lampung dan sedang menempuh pendidikan di Universitas Muhammadiyah Kotabumi pada Program Studi Sistem dan Teknologi Informasi. Saya memiliki beberapa hobi, di antaranya jalan-jalan, memasak, dan makan, serta warna favorit saya adalah pink dan biru. Dalam hal kepribadian, saya cenderung introvert, namun saya akan menjadi lebih terbuka dan mudah berinteraksi ketika sudah merasa nyaman dengan lingkungan maupun orang-orang di sekitar saya.
